style을 어떻게 적용 시킬까?

1. html 태그에서 직접 적용시킬 수 있다.

이건 style 적용 안 한 것.

이건 style 적용 한 것.

2. <head></head> 태그 안에 <style></style> 태그에서 적용시킬 수 있다.

먼저 html 코드 구조를 봐보자

코드를 보니<html></html>태그 안에 <head></head>라는 태그와 <body></body>라는 태그가 보인다.

<head></head>라는 태그는 메타 데이터를 포함하고

<body></body>라는 태그는 화면에 그려지는 요소를 포함한다.

<head></head> 태그 안에 <style></style> 태그를 찾아 볼 수 있다.

<style></style> 태그 사이에 코드를 입력하여 여러 요소들에 style을 적용시킬 수 있다.

다음은 기본 HTML 코드 예제입니다:

VSCode에서 !표를 입력해 봅시다. 기본 HTML 코드가 자동 입력됩니다.


<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>HTML Code Snippet Example</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is a paragraph.</p>
</body>
</html>
